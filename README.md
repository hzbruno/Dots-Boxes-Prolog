This project was developed as part of a university course on Logic Programming, focusing on the use of Prolog as the main programming language. 
The goal was to design and implement a fully functional version of the classic game Dots and Boxes, integrating logic-based reasoning with a web interface.

The core logic of the game — including board representation, move validation, game rules, and AI behavior — is implemented entirely in Prolog. 
The project also includes a web server interface built using TauProlog libraries, allowing the game to be played directly from a web browser. 
This setup demonstrates how logical inference and declarative paradigms can interact with modern web technologies.

Key features of the project include:
  A rule-based engine for handling game mechanics.
  Support for player vs. player and player vs. computer modes.
  A simple yet functional user interface accessible via a web browser.
  Modular and well-documented Prolog code for easy maintenance and extension.

This project showcases the practical application of Prolog beyond theoretical exercises, illustrating how logic programming can be used to build interactive applications.
